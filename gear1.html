How to Play

🎯 Goal: Connect the GREEN gear to the RED gear using all available gears!

🖱️ Drag gears from the left tray to the game area.

⚡ Power: Click the Power button to start the gears spinning!

🔗 Connect: Place gears close to each other - they will snap together!

🌟 Tip: Use all gears and make sure power reaches the red gear to win!

🎵 Sound: Toggle sound on/off using the speaker button.
